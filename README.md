# Lia
Lia: The Unified AI Operating Environment
Lia is far more than a standard AI chatbot; it is a meticulously designed, comprehensive operating environment. Wrapped in a stunning, high-performance dark-mode UI with real-time system monitoring, Lia unifies local AI interaction, advanced code editing, bot management, and content creation into a single, seamless application.

Designed for power users, developers, and creators, Lia gives you total control over your local and cloud-based models while providing the tools to automate, build, and manage your digital presence.

🌟 Core System Features
The foundation of the Lia application is built on performance and control.

Real-Time System Telemetry: Live monitoring of CPU usage, RAM usage, and system temperature directly within the sidebar.

Dynamic Memory Management: Persistent memory controls allowing you to edit, clear, or export Lia's conversational memory at any time.

Multi-Modal Inputs: Seamlessly transition between text, voice recognition, and web-connected context tools.

🛠️ The Modules
1. Chat with Lia
The primary conversational hub designed for fluidity and focus.

Model Agnostic: Switch between different local models (via Ollama) or cloud models on the fly.

Vibe Modes: Engage distinct behavioral states, such as the active Chill Mode.

Quick Tools: Access to web searches, voice-to-text, audio playback, and thought-process toggles.

2. Code IDE
A fast, built-in scratchpad for rapid development and testing.

Live Preview: Instantly write and preview code (e.g., HTML Live Preview).

Execution Environment: Run code directly within the UI, complete with STDIN inputs and a dedicated console output window.

Formatting: Built-in code formatting for clean, readable scripts.

3. Studio (Agentic Workspace)
A full-scale, integrated development environment powered by the Lia Agent.

Pro-Mode AI Coder: Lia acts as an agentic partner that plans before coding, works across multiple files simultaneously, and provides diff-previews before applying changes.

Project Explorer: Full file and folder navigation to manage complex Python projects.

Integrated Terminal: Run shell commands, track Git logs, and monitor terminal outputs without leaving the app.

4. Discord Bot Command Center
A complete, enterprise-grade dashboard for managing your custom Discord bot.

Live Controls: Start, stop, and fully configure the bot server with a click.

Module Management: Toggle Command-Only modes, DM Auto-Replies, and bot-to-bot interactions.

Deep Customization: Manage everything from Server Economy and Moderation to AI Settings, customized 'Feelings', and Broadcast messaging.

5. Creator Studio
An automated powerhouse for YouTube content creation.

Local TTS Engine: Generate high-fidelity, long-form voiceovers utilizing fast, local TTS engines like Piper and Kokoro.

End-to-End Generation: Automate the creation of YouTube scripts, titles, descriptions, hooks, chapters, and hashtags.

AI Vision: Integrated tools to assist with thumbnail conceptualization and visual generation.

6. Telegram Bot Dashboard
A parallel command center dedicated to your Telegram ecosystem.

Fleet Management: Control bot uptime, DM auto-replies, and command-only states.

Quick Messaging: Send direct messages or broadcasts to specific Chat IDs via markdown directly from the UI.

Community Tools: Manage Group Stats, Anti-Spam rules, Welcome messages, and channel syncing.

7. Unified Logs
The ultimate diagnostic and tracking tool.

Cross-Platform Feed: Monitor requests, warnings, and system logs across all of Lia's modules (Discord, Telegram, Local IDE) in one place.

Exportable Data: Pull 30-day reports and export logs seamlessly to JSON or CSV for external analysis.

## 🌸 Installation

### Step 1 — Install Dependencies
Download `install_lia.sh` from the releases page and run it:

```bash
chmod +x install_lia.sh && ./install_lia.sh
```

### Step 2 — Download Lia
Go to the [Releases](https://github.com/TechnoTechYT/Lia-releases/releases)
page and download the latest version for your platform.

### Step 3 — Extract the archive
```bash
tar -xzf Lia-1.0.0-linux.tar.gz
```

### Step 4 — Launch
Simply **double-click the `Lia` executable** inside the extracted folder — 
no terminal needed. Lia will open automatically in your browser. 🌸

> **Note:** Run `install_lia.sh` **before** launching Lia for the first 
> time, otherwise some features may not work correctly.
